・Altキーを押しながら移動すると速く移動できる
・マップの開始時間・プレイ時間・時間を追加

・芝生・木の色を変更
・マップファイルの形式を変更（以前のマップは使用不可）
・Shiftを押しながら移動するとカメラを高速移動できるようになった
